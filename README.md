# «Неткоин» - сайт-биржа для крипто-стартапа

На этом сайте можно покупать внутреннюю валюту компании **Неткоин** и передавать эту валюту другим пользователям сервиса. 

Веб-приложение для управления внутренней валютой Неткоин: регистрация и вход пользователя, просмотр профиля, пополнение баланса, перевод средств другим пользователям и работа со списком избранных получателей.

Проект выполнен как дипломная работа по курсу «Основы JavaScript». Основной акцент — на клиентской логике: обработке пользовательских действий, взаимодействии с API, обновлении интерфейса на основе ответа сервера и корректной обработке ошибок.

## Установка и запуск

<br>

### Настройка сервера

1. Установите [NodeJS](https://nodejs.org/en/) для своей операционной системы.
2. Клонируйте репозиторий: https://github.com/potykalov/bjs-diplom.git.
3. Откройте терминал и перейдите в папку с клонированным репозиторием.
4. Установите зависимости, выполнив команду `npm install`.

Убедитесь, что выполняете команды с правами администратора.

### Запуск сервера

1. Откройте терминал, перейдите в папку с клонированным репозиторием и выполните команду `npm start`. Убедитесь, что в консоли появилась надпись `Server started at 8000`.
2. Перейдите по [ссылке](http://localhost:8000). Вы увидите интерфейс Биржи Неткоинов.

### Завершение работы сервера

1. Если нужно прекратить работу сервера, нажмите комбинацию клавиш `Ctrl+C`.
